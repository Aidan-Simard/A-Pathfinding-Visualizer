# Astar-Pathfinding-Visualizer

A* pathfinding visualizer using pygame.

## How to install & use the visualizer:

### Requirements:
- This visualizer uses Pygame version 2.0.1 & Python version 3.8.6.
- If you do not have Pygame installed, run `pip install pygame==2.0.1` in your terminal.

### Using the visualizer:
- Open your terminal and locate the installation path.
- Run the visualizer with `python astar_pathfinder.py`.
- The first click corresponds to the "starting node", the second click corresponds to the "ending node".
- Then, click and drag to create walls between said nodes.
- Finally, press the space bar to execute the pathfinding algorithm.
